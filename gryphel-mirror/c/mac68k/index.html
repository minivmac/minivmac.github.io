680x0 Macintosh

List

Here is a some information about 680x0 Macintosh models that could be useful to emulator users. It may not be too accurate, it is just my best guess from gathering information from many conflicting sources.

This list is grouped by ROM version (different Macintosh models can share the same ROM), and sorted by initial sale date. "Rom Image Name" is the name of the file that my CopyRoms program will save. It is also the name of the file that Mini vMac will look for, if Mini vMac supports that ROM (in which case the entry includes the line "Supported by Mini vMac: yes"). To emulate any machine other than a Mac Plus, Mini vMac needs to be recompiled (See the "Options" documentation).

The "ROM Checksum" is stored in the first 4 bytes of the ROM. It is usually a simple 32 bit sum of all 16 bit values in the ROM, excluding the checksum. A Macintosh will compute this sum on startup to help make sure the ROM is undamaged. (Since Mini vMac modifies the ROM, it will first make this check itself, and then disable the checking code in ROM.)

I have also provided a "ROM md5 Checksum" where I know it, which can provide a much stronger assurance that you have an undamaged ROM. Numerous programs can compute an md5 checksum. I provide the program Md5Im for computing it using Mini vMac.
